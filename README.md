# 🔭 ORBITAL-CHOHRA — Real-Time Web Observatory

**The first free, real-time, French-language web space observatory — built from Tlemcen, Algeria.**

🌐 **Live**: [astroscan.space](https://astroscan.space)

## Features
- 🛸 ISS Live Tracking + Pass Predictions (SGP4/TLE)
- 🌍 3D Orbital Globe — Cesium.js (1,000+ satellites)
- 🌞 NOAA Space Weather Alerts + Kp Index
- 🔭 NASA APOD, JWST, Hubble, Mars Rovers
- 🚀 Voyager 1 & 2, Parker Solar Probe, DSN Live
- ☄️ NASA NEO — Near-Earth Asteroids
- 🧠 AEGIS AI Chatbot (Gemini) in French
